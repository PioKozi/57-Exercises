# Guess the Number Game

Write a Guess the Number game that has 3 levels of difficulty. The first would
be a number 1-10, second 1-100, third 1-1000.

**EXAMPLE:**

```plaintext
Pick a difficulty: 1
What's your guess? 1
Too low
What's your guess? 5
Too high
What's your guess? 2
You got it in 3 guesses!
Play again? n
```
